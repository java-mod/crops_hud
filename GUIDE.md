# 작물 수확 세션 HUD — 사용 가이드

> Minecraft 1.21.4 / Fabric
> 클라이언트 전용 모드

---

## 개요

작물을 수확하는 동안 **수확량, 특수 드랍, 활성 시간, 예상 수익, 시간당 수익**을 화면 HUD로 실시간 표시합니다.
가격을 직접 설정하면 수확한 양을 기준으로 수익을 자동 계산합니다.

---

## HUD 표시 항목

| 항목 | 설명 |
|------|------|
| **작물** | 현재 수확 중인 작물 이름 (감지 전: "감지 대기 중") |
| **수확량** | 이번 세션에서 수확한 일반 작물 총 개수 |
| **특수** | 서버 특수 드랍 아이템 감지 횟수 (1개 이상이면 황금색 강조) |
| **활성 시간** | 실제로 수확한 시간 (유휴 시간 제외) |
| **예상 수익** | 수확량 × 설정 가격 |
| **시간당 수익** | 예상 수익 ÷ 활성 시간(시간 단위), 활성 시간이 짧으면 `--` 표시 |

- 상태 뱃지: **수확 중** (초록) / **일시정지** (베이지)

---

## 세션 동작 방식

### 자동 시작
작물을 수확하는 순간 세션이 자동으로 시작됩니다. 별도 명령 불필요.

### 자동 일시정지
마지막 수확으로부터 **5초** 동안 아무것도 수확하지 않으면 자동으로 일시정지됩니다.
다시 수확하면 자동으로 재개됩니다.

### 자동 초기화
**다른 종류의 작물**을 수확하면 이전 세션이 자동으로 초기화되고 새 세션이 시작됩니다.

### 인식하는 작물 조건
수확으로 인정되려면 아래 조건을 만족해야 합니다.

| 작물 | 조건 |
|------|------|
| 밀, 당근, 감자, 비트루트 | **완전히 성장한** 상태(`age` 최대)일 때만 카운트 |
| 네더와트 | 완전히 성장한 상태 |
| 코코아콩 | 완전히 성장한 상태 (`age` = 2) |
| 사탕수수, 선인장 | 밑에 같은 블록이 있는 단(2단 이상)만 카운트 |
| 호박, 수박 | 블록 그대로 부수면 카운트 |

## 특수 드랍 감지

서버 플러그인이 지급하는 특수 작물 아이템을 인벤토리 모니터링으로 자동 감지합니다.
수확 직후 **1.5초 이내**에 인벤토리에 들어온 특수 아이템만 집계하여 외부에서 아이템을 받아도 잘못 카운트되지 않습니다.

| 수확 작물 | 특수 드랍 아이템 | 식별 이름 |
|-----------|----------------|----------|
| 밀 | 빵 | 구수한 밀 |
| 당근 | 당근 (커스텀 이름) | 아삙한 당근 |
| 감자 | 구운 감자 | 포슬한 감자 |
| 비트루트 | 비트루트 (커스텀 이름) | 달콤한 비트 |
| 네더와트 | 말린 다시마 | 알싸한 네더 사마귀 |
| 코코아콩 | 쿠키 | 향긋한 코코아콩 |
| 수박 | 수박 조각 | 아삙한 수박 |
| 호박 | 호박 파이 | 달달한 호박 |

> 사탕수수, 선인장은 특수 드랍 없음

## 명령어

### 작물 가격 설정 — `/작물가격` (또는 `/cropprice`)

#### 가격 설정
```
/작물가격 설정 <작물> <가격>
/작물가격 설정 밀 12.5
/작물가격 설정 당근 8
```

#### 가격 조회
```
/작물가격 조회 <작물>
/작물가격 조회 밀
```

#### 전체 목록
```
/작물가격 목록
```

---

### HUD 관리 — `/허드` (또는 `/hud`)

#### 세션 초기화
```
/허드 초기화
```
수확량, 시간, 수익을 모두 0으로 리셋합니다.

#### HUD 위치 변경 (앵커 프리셋)
```
/허드 위치 <위치>
```

사용 가능한 위치:

| 값 | 설명 |
|----|------|
| `좌상단` / `top_left` | 좌측 상단 |
| `우상단` / `top_right` | 우측 상단 **(기본값)** |
| `중앙상단` / `center_top` | 상단 중앙 |
| `중앙` / `center` | 화면 중앙 |
| `좌하단` / `bottom_left` | 좌측 하단 |
| `우하단` / `bottom_right` | 우측 하단 |
| `중앙하단` / `center_bottom` | 하단 중앙 |

예시:
```
/허드 위치 우하단
/허드 위치 bottom_right
/허드 위치 중앙상단
```

#### HUD 위치 직접 조정 (드래그)
```
/허드 편집
```
HUD 카드를 마우스로 드래그하여 원하는 위치에 자유롭게 배치할 수 있습니다.
화면 가장자리 및 중앙에 **스냅 가이드**가 있어 정렬이 쉽습니다.
**ESC** 를 누르면 위치가 저장됩니다.

키 바인딩 설정(`설정 → 조작 → 작물 수확 HUD`)에서 단축키를 지정하면 명령어 없이도 편집 화면을 열 수 있습니다.

HUD 위치는 `config/crophud/crophud_hud.json`에 자동 저장됩니다.

---

## 팁

- 가격을 설정하지 않은 작물은 **예상 수익이 0**으로 표시됩니다. 먼저 `/작물가격 설정`으로 가격을 등록하세요.
- 서버마다 좌상단에 커스텀 HUD가 있을 수 있으니, 겹칠 경우 `/허드 편집`이나 `/허드 위치`로 위치를 조정하세요.
- 세션은 서버 연결이 끊기면 자동으로 초기화됩니다.
- `/수확세션`, `/harvestsession` 명령어는 `/허드`의 구 별칭으로 하위 호환됩니다.
